¡Instrucciones Importantes!

Para asegurar una importación exitosa, por favor, sigue estos pasos:

  1. Descarga nuestra plantilla oficial. Esto garantiza que las columnas tengan el formato y los nombres correctos.
    Descargar Plantilla de Importación
  2. Llena la plantilla con los datos de los aspirantes.
    • La columna 'grado_aspira_id' debe contener el ID numérico del grado (disponible en la hoja "Ayuda" de la plantilla).
    • ⭐ Nuevo campo obligatorio:
      Agrega la columna paga_inscripcion (SI/NO) en tu archivo.
      • Escribe SI, SÍ, TRUE o 1 para que el sistema genere la cuenta por pagar.
      • Escribe NO, FALSE o 0 para evitar generar el cobro.
      • Si el campo está vacío o contiene un valor no válido, el registro será rechazado.
  3. Guarda el archivo y súbelo usando el formulario de abajo.
